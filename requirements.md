# Requirements

## Project Name

AI Personal Textbook Generator

## Purpose

AWS Certified AI Practitionerの学習教材を自動生成する

## Target User

- AWS Certified AI Practitioner受験者

## MVP

- 資格名を入力できる
- 試験範囲を取得できる
- Webから情報を収集できる
- 教材をMarkdown形式で生成できる

## Out of Scope

- ユーザー管理
- 課金機能
- SNS機能
- 複数資格対応

## Success Criteria

AWS Certified AI Practitioner向けの教材を1冊生成できること
